#### blog-reactjs-ts


Used this libraries:

- ReactJS 
- NextJS
- TypeScript
- Redux
- Redux Saga
- Ant Design
- Axios
- Prettier


### Running project

To run this project execute commands:

```
npm intall
npm run dev
```

Run prettier 

```
npm run format
```

The blog has routes:

```
/
/posts/:id
/posts/new
```
